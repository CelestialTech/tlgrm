Tlgrm - Кастомный Telegram Desktop
====================================

Инструкция по установке:
1. Перетащите Tlgrm.app в папку Программы
2. Откройте Tlgrm из папки Программы
3. Войдите с помощью номера телефона

Возможности:
- Полный функционал Telegram Desktop
- Кастомная сборка, оптимизированная для macOS
- Универсальный бинарник (Intel + Apple Silicon)
- Поддержка macOS Ventura 13.0 и выше

Важные заметки:
- Это кастомная сборка Telegram Desktop
- Работает независимо от официального приложения Telegram
- Данные хранятся в ~/Library/Application Support/Tlgrm
