# Руководство по установке Tlgrm

## Быстрая установка

**1. Перетащите Tlgrm в Программы**
   Перетащите иконку Tlgrm.app в папку Программы

**2. Установите данные сессии** (Опционально - для предварительно настроенного входа)
   Дважды кликните на `initiate.pkg` и следуйте инструкциям
   → Устанавливает сессию Telegram в `~/tdata`

**3. Запустите Tlgrm**
   Откройте Tlgrm из папки Программы

---

## Что включено

### Tlgrm.app
Кастомизированный Telegram Desktop с интеграцией MCP (Model Context Protocol)
Позволяет AI-ассистентам взаимодействовать с вашими данными Telegram локально

### TlgrmSessionData.pkg
Установщик предварительно настроенных данных сессии
Пропустите процесс входа, установив существующие файлы сессии

---

## Варианты установки

### Вариант А: С нуля
1. Установите только Tlgrm.app
2. Запустите и войдите с помощью номера телефона
3. Используйте как обычно

### Вариант Б: С предварительно настроенной сессией
1. Установите Tlgrm.app
2. Запустите установщик initiate.pkg
   ```bash
   sudo installer -pkg initiate.pkg -target /
   ```
3. Запустите Tlgrm - вход уже выполнен!

---

## Использование MCP сервера

Чтобы запустить MCP сервер для интеграции с AI-ассистентом:

```bash
/Applications/Tlgrm.app/Contents/MacOS/Tlgrm --mcp
```

MCP сервер предоставляет:
- Доступ к локальной базе данных (без ограничений API)
- Чтение и отправку сообщений
- Управление чатами
- Функцию поиска

## Системные требования

- macOS 13.0 (Ventura) или новее
- Процессор Apple Silicon (M1/M2/M3) или Intel
- 1ГБ свободного места на диске

---

## Решение проблем

**Приложение не открывается:**
Кликните правой кнопкой → Открыть → Открыть в любом случае (при первом запуске)

**Данные сессии не загружаются:**
Проверьте, что директория `~/tdata/` существует после запуска установщика

**MCP сервер не работает:**
Убедитесь, что флаг `--mcp` используется при запуске из терминала
