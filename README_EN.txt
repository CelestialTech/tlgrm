Tlgrm - Custom Telegram Desktop
================================

Installation Instructions:
1. Drag Tlgrm.app to the Applications folder
2. Open Tlgrm from Applications
3. Log in with your phone number

Features:
- Full Telegram Desktop functionality
- Custom build optimized for macOS
- Universal binary (Intel + Apple Silicon)
- Supports macOS Ventura 13.0 and later

Important Notes:
- This is a custom build of Telegram Desktop
- Works independently from official Telegram app
- Data is stored in ~/Library/Application Support/Tlgrm
